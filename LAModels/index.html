Linear Algebra Models

LAModels is a module that provides a minimum interface needed by different linear algebra problems. This interface includes abstract types to specify the type of problem being considered, and it includes test to ensure that any subtypes of the abstract types contain the minimum necessary interface.

Abstract Types

Main.LAModels.LinearSystemsProblem — Type
LinearSystemsProblem

Abstract type to model a consistent linear systems problem,

\[Ax = b,\]

where $x$ is unknown.

source
Main.LAModels.LeastSquaresProblem — Type
LeastSquaresProblem

Abstract type to model a least squares problem,

\[\min_{x} \Vert Ax - b \Vert_2^2.\]
